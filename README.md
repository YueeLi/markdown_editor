# Markdown Editor

轻量级 Markdown 在线编辑器，满足以下需求：
- 顶部工具栏：导入、导出、复制、清空、左右切换
- 主体左右分栏：左侧编辑、右侧实时预览（可一键左右切换）
- 纯前端实现：可离线使用，自动本地保存（localStorage）

## 在线运行
本项目为纯静态站点，可直接部署到 Vercel 或任何静态托管。
- 入口文件：`index.html`
- 依赖通过 CDN 注入：`marked`（Markdown 解析）、`DOMPurify`（安全净化）

## 功能说明
- 导入：支持 `.md/.markdown` 文件导入（文件选择、拖拽）
- 导出：将当前内容保存为 Markdown 文件（自动根据首个标题生成文件名）
- 复制：复制当前 Markdown 文本到剪贴板
- 左右切换：互换编辑与预览面板位置（也支持移动端纵向布局）
- 清空：清空当前内容（带确认提示）
- 实时预览：每次输入即更新预览，HTML 经过 DOMPurify 保护避免 XSS
- 本地自动保存：内容会保存到浏览器 `localStorage`，刷新不丢失

## 部署到 Vercel（手动）
1. 新建 Vercel 项目，选择此仓库或手动上传
2. 框架选择 `Other`（纯静态）
3. 构建命令留空；输出目录设置为根目录 `/`

## 本地开发
无需构建步骤，直接双击 `index.html` 即可运行。也可使用轻量 HTTP Server：
```bash
python3 -m http.server 8080
# 然后访问 http://localhost:8080
```

## 代码结构
```
/markdown_editor
├── index.html   # 页面结构 & 资源引用
├── style.css    # 布局与视觉样式
├── script.js    # 交互逻辑（导入/导出/复制/切换/预览）
└── README.md    # 文档
```
